锤子手机Pro2刷机笔记

发表于 2019-07-28 更新于 2019-08-17 分类于 安卓手机

锤子手机坚果Pro2刷机

1. 刷机准备

锤子手机是之前的名称，老罗把公司迁到成都后，锤子在四川方言中是骂人的话，就改成了坚果手机。我手里的手机既是坚果Pro2，在2017年九月发布之后买的。颜值和配置都是杠杠的。

无奈，随着使用，手机逐渐变的卡顿，遂萌发刷机换系统的想法，需要准备的东西如下：

  1. 高通 EDL 线（也称为 9008 线、救砖线、工程线），自行上淘宝购买（10几块的就够了，反正就用一次。淘宝上有些卖家会写着「小米工程线」，其实是同一个东西，通用的）；

  2. QPST 线刷工具、TWRP 线刷包、底包、魔趣 ROM 等相关资源，详见：

    • 普通版 (OS105)：坚果手机 Pro 2 (osborn) 底包、TWRP 及相关工具
    • 特别版 (OS103)：坚果手机 Pro 2 特别版 (oxford) 底包、TWRP 及相关工具

  3. 备份重要数据

刷机有风险，请小心谨慎，强烈建议在刷机前备份重要数据如联系人、短信、照片、视频等重要文件。

2. 开始刷机

2.1 刷入 TWRP Recover 底层引导

TWRP 官网：https://twrp.me/

坚果锤子手机官网并没有提供，我们使用的都是各路大神更改过的。

因为坚果Pro2存在底层设备锁（官方为了系统的安全稳定设置的一道安全工具，可以防止root）我们在刷第三方 Recover 的时候需要整一根9008工程线。

具体步骤参考油管视频：https://www.youtube.com/watch?v=sbmU08MqxZA

文件链接: https://pan.baidu.com/s/1BMNTtnqrRrKmdh5NitqbVA 1.4K 提取码: 6j4e

具体步骤如下：

  1. 安装 QPST 工具包中的线刷工具和驱动，打开 Qfil 软件；
  2. 进 Configuration，确保 Device Type 选了 eMMC；
  3. 拔掉数据线， 关机。等待手机彻底关机、屏幕不亮为止；
  4. 把 EDL 线插到电脑上，按住 EDL 线上的小开关把另一头插入手机。等待大概 3 秒松手，此时电脑上的 Qfil 应该会识别到 Qualcomm HS-USB QDLoader 设备；
  5. 在 Qfil 里选择 Flat Build；
  6. 点击 Programmer 的那个 Browse，选择线刷包中的 prog_ 开头的文件；
  7. 点击 Load XML，选择工具包里的 rawprogram_unsparse.xml；
  8. 它会再次弹出选择框，点取消；
  9. 深呼吸；
  10. 点击 Download 按钮，稍等片刻，TWRP 刷入完成；
  11. 长按音量减和电源键。直到你看到白色锤子的时候松开电源键（继续按住音量减键）；直到你看到 TWRP 的时候才松开所有按键，进入 TWRP，继续进行下面的步骤。

如果刷机失败，或打算中途放弃，你可以按住音量加和电源键退出 EDL 模式，回到正常系统。

2.2 刷入底层包及魔趣系统

底层包的作用主要是提供系统基带等底层应用。

  1. 进 Wipe（清除），把下面那个条条拖住向右划一下
  2. 返回上一层，进 Install（安装），刷入底包（RADIO- 开头的那个）
  3. 返回上一层（不要重启），再次进安装，刷入魔趣（MK 开头的那个）
  4. 刷完重启，完成

最新魔趣安卓 9.0 谷歌原生系统下载链接：https://download.mokeedev.com/?device=osborn

2.3 刷入谷歌框架

有些同志想要体验 Google 服务（需要自备科学上网环境），下载相应文件：

魔趣论坛 GAPP 下载链接：https://download.mokeedev.com/extension.html

  1. 下载 open_gapps-arm64-9.0-nano-20190727.zip 放到内置存储根目录;
  2. 重启进入恢复模式;
  3. 点击 install 安装；
  4. 重启进入系统；
  5. 搞定。

3. 系统升级

魔趣中心自带了系统在线升级方式，捐赠后可以使用增量升级，相当方便。不过需要注意的是：相应的系统有其对应的系统底包，只有底包与系统一致时才可以升级成功，所以可以是不是去魔趣官网看看是不是又出新的系统底包了。

4. 刷其他版本系统

在使用一段时间后，如果想体验其他版本的系统，可以在 TWRP 引导程序的基础上直接刷：

  1. 下载相应的底包与刷机包
  2. 手机关机状态下按住音量下键/上建 + 电源键，待屏幕点亮后松开电源键，进入 TWRP Recover 引导程序
  3. 选择 Wipe 滑动底部滑动条进行系统双清，。不要重启系统
  4. 返回选择 Install，刷入底包，不要重启系统
  5. 返回选择 Install，刷入系统包即可。

5. 如何刷回原厂 Smartisan OS 系统

  1. 下载官方卡刷包放到内置存储根目录；
  2. 在 TWRP 里 Wipe，然后刷入卡刷包。

官方下载链接：6.7.0.1-201906111921-user-ob-cccc8e395b.zip
